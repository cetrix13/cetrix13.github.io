It is happened, I'm trying out Jekyll to set up a personal blog! Github's tool "Pages" is used as hosting and I write
articles in markdown language. We will see how it goes to use on daily basis.

<br>
Right now I'm kind of suprised how easy it was to set up a platform, push changes to the
repository and have it to go online. Just a few hours mostly to have look and feel of how I wanted blog to appear.
<br><br>
Used this chance to learn how to use Bulma CSS framework. It uses flexbox and was easy to customize, fortunately Jekyll
can compile .sass files on the fly. Probably should use this opportunity to learn how to write styles in .sass format
rather than .scss which I am more familier.
<br>
There are some work to do, though it is a good starting point.
